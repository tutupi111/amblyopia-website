# 弱视儿童康复训练网站（Amblyopia Training Web）

这是一个面向弱视儿童家庭与治疗师的康复训练网站原型，包含：

- 康复训练首页与家长提示
- 训练任务管理（本地存储版）
- 6 个可直接体验的小游戏：
  - 对比敏感度点击训练
  - 视动追踪训练
  - 视觉记忆卡片训练
  - 红蓝眼镜分视融合对齐训练
  - 红蓝眼镜分视双眼接球训练
  - 方块拼接训练（Tetris-like）
- 训练记录与连续打卡统计

## 快速开始

直接在浏览器打开 `index.html` 即可。

> 当前版本为前端单页原型（无后端）。数据保存在浏览器 `localStorage`。

## 建议下一阶段（生产可用）

1. 引入账户体系（家长/治疗师/儿童）
2. 增加训练处方引擎（按年龄、视力阶段、双眼协调能力）
3. 接入云端数据库与报告导出
4. 增加可配置游戏参数与疗效分析

## 文件结构

- `index.html`：页面结构
- `styles.css`：样式与无障碍视觉优化
- `app.js`：训练逻辑、游戏逻辑、数据统计

## 部署 404 排查（Vercel）

如果访问时出现 `404: NOT_FOUND`，通常是静态站点路由未回退到 `index.html`。

本仓库已新增 `vercel.json`，将所有路径重写到 `index.html`，用于单页应用部署。
