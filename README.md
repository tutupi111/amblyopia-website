# 弱视儿童康复训练网站（Amblyopia Training Web）

本版本重点改为“**游戏大厅 + 独立游戏页弹层**”模式，避免多个小游戏同时展示造成控制干扰。

## 当前能力

- 游戏大厅按训练目标分类：
  - **增视训练**：动态黑白条栅、对比敏感、视动追踪、记忆、方块拼接
  - **双眼视觉训练**：红蓝眼镜分视（融合对齐、双眼接球）
- 每个小游戏点击后进入独立弹层页面
- 修复方向键影响页面滚动的问题（方块训练时拦截箭头/空格默认滚动行为）
- 本地任务和训练统计（`localStorage`）

## 快速开始

直接在浏览器打开 `index.html`。

## 部署

- 已提供 `vercel.json`，将所有路径回退到 `index.html`，避免 SPA 场景 404。
